# Contribuir a Free Claude Config

Opciones de contribución y mejoras de documentación.

## Cómo contribuir
1. Haz fork del repositorio.
2. Crea una rama con nombre claro:
   ```powershell
git checkout -b docs/mejoras
```
3. Añade documentación, correcciones o scripts útiles.
4. Envía un pull request hacia `master`.

## Buenas prácticas
- Trabaja en un entorno virtual.
- No incluyas claves ni archivos locales.
- Mantén la documentación actualizada y precisa.
